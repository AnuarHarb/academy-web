micro curso React

Inicio

19 de Marzo.

Al salir podrás:

Podrás realizar aplicaciones SPA usando react que cumplan con las siguientes características: Múltiples rutas, implementación de autenticación y rutas protegidas, manejo de listas y filtrado de datos, conexión entre componentes, ventanas modales y manejo de errores en componentes. Además de su correspondiente montado en Firebase Hosting

Objetivo

Utilizar React para el desarrollo de aplicaciones Single Page Apps del lado del cliente y entender cómo funciona el desarrollo de aplicaciones usando componentes para agilizar el proceso de construcción de aplicaciones frontend.

Perfil del aspirante

Conocimiento de JS (objetos, promesas, funciones, closures) y manejo de módulos en NodeJS
