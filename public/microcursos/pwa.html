micro curso PWA

Inicio

Abril

Objetivo

Generar aplicaciones que puedan utilizarse de manera offline y puedan hacer uso de notificaciones push.

Perfil del aspirante

Conocimiento de JS (objetos, promesas, funciones, closures) y manejo de módulos en NodeJS

Al salir podrás:

Entenderás el proceso para crear una PWA desde 0 y lo que implica hacer una migración de una aplicación normal a una PWA sin importar la tecnología con lo que esté construida. Podrás brindar soporte offline y entender los tipos de estrategia para cargar contenido que pueden brindar una mejor experiencia a tu usuario, además de entender la infraestructura necesaria para brindar de soporte para notificaciones push.

Duración y costos

Plan de Estudios

+ Clase 1

  • Definir ¿Qué es una Progressive Web App?
  • Crear un archivo manifest para una aplicación web
  • Definir el App Sheel de una aplicación
  • Controlar la aparición del banner de instalación de una PWA
  • Entender las funciones básicas de un service worker: funcionalidad, ciclo de vida y revisión en panel de Application de chrome

+ Clase 2

  • Implementar un service worker
  • Suscribir un service worker al evento fetch
  • Actualizar un service worker
  • Implementar distintos tipos de estrategia de red dependiendo del tipo de recurso
  • Hacer uso de Cache API e IndexedDB

+ Clase 3

  • Hacer un overview de la funcionalidad de workbox
  • Implementar estrategias red y app shell usando workbox en modo CLI
  • Se hará uso de Workbox usando Webpack

+ Clase 4

  • Entender el funcionamiento del API de notificaciones y el Push Manager
  • Entender cómo funciona una arquitectura para realizar notificaciones push
  • Implementar un service worker que sea capaza de recibir notificaciones push
  • Obtener token de suscripción
  • Hacer uso de un servidor que mande notificaciones push a un dispositivo

+ Clase 5

  • Entender cómo funciona el evento sync
  • Implementar sincronización
  • Hacer deployment a Firebase
