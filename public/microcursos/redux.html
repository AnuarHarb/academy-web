micro curso Redux

Inicio

25 de Marzo.

Objetivo

Comprender el flujo de Redux y cómo podemos utilizarlo para construir aplicaciones más robustas sin preocuparnos por la administración del estado de la aplicación.

Perfil del aspirante

Conocimiento de JS (objetos, promesas, funciones, closures) y manejo de módulos en NodeJS

Al salir podrás:

Al finalizar podrás entender la utilidad de redux e implementarlo en aplicaciones a vanilla o frameworks como react, entenderás el flujo de react y cómo administrar el estado de la aplicación, aplicarás conceptos de programación funcional para crear una arquitectura sólida para administrar el store de tu aplicación con redux.

Plan de Estudios

+ Clase 1

  • Entender conceptos base de Redux y como se manipula el estado de una aplicación
  • Instalar herramientas para observar el estado creado por redux
  • Entender conceptos de programación funcional y como se aplican estos a redux
  • Comprender el flujo de redux mediante ejercicios y testing
  • Crear una biblioteca que “funcione” como Redux
  • Hacer una implementación Redux

+ Clase 2

  • Crear constantes para actions names y funciones que los usen para generar una acción
  • Delegar las ramas del store a varios reducer para hacer código más legible usando combineReducers
  • Entenderás el concepto de composición en programación funcional
  • Entender donde se ejecutan los middleware, su importancia y cómo pueden implementarse

+ Clase 3

  • Agregar datos al estado en una operación asíncrona de manera manual (conexión con una API)
  • Agregar datos al estado usando acciones asíncronas (conexión con una API)
  • Entender el funcionamiento básico de un generator y su modo de ejecución
  • Implementar redux-saga (conexión con una API)

+ Clase 4

  • Hacer una implementación de un High Order Reducer para reutilizar lógica en diferentes ramas del store
  • Implementar testing a acciones y reducers y su efecto sobre el store
  • Conectar React con Redux

+ Clase 5

  • Entender el concepto de memoization
  • Implementar Redux Selector
  • Implementar llamada a API y conectarla con Redux y React
